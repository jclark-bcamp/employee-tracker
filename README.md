  # Employee Tracker

  ## Description
  This Employee tracker creates a table of Departments that include various data including Employees, Managers, and Salaries.

  ## Installation
  Inquirer

  ## Usage
  Once you have both npm packages installed, open Node and run the application. You will be prompted with various options, including Adding or Viewing data in the table.
